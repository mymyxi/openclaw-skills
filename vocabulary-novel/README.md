## 🌟 项目介绍

**Vocabulary Novel** 是一个创新的英语学习技能，通过**沉浸式小说阅读**帮助学习者自然掌握英语词汇。

### 核心理念

> **剧情第一，单词第二**

我们相信最好的学习是**不知不觉发生的**。与其枯燥地背单词书，不如沉浸在精彩的故事中，让单词自然融入记忆。

### 传统背词 vs 词汇小说

| 传统背词 | 词汇小说 |
|---------|---------|
| ❌ 枯燥重复 | ✅ 精彩剧情吸引你继续阅读 |
| ❌ 脱离语境 | ✅ 单词在自然场景中出现 |
| ❌ 容易遗忘 | ✅ 科学复现，符合记忆曲线 |
| ❌ 难以坚持 | ✅ 想知道后续，不知不觉学完 |

## 🛠️ 使用方法

### 作为读者

1. **选择词库**：根据目标考试选择对应词库
2. **开始阅读**：像普通小说一样阅读
3. **自然学习**：单词在语境中出现，自然记忆
4. **复习巩固**：可配合词库表复习

### 作为创作者

1. **阅读 SKILL.md**：了解写作规范
2. **选择词库**：从 `vocabulary/` 目录选择词库
3. **遵循大纲**：每章有指定新词和复现词
4. **自然融入**：使用正确的单词融入技巧
5. **检查复现**：确保单词按记忆曲线复现

## 🎯 单词融入技巧

### ✅ 正确示范

**情感描写**
```
我感到一阵 panic（恐慌），手心开始出汗。
心里莫名地 nervous（紧张），不知道会发生什么。
```

**环境描写**
```
他穿着 elegant（优雅的）黑色西装，气质出众。
房间装饰得 gorgeous（华丽的），像宫殿一样。
```

**动作描写**
```
我 freeze（僵住）在原地，大脑一片空白。
他 stare（凝视）着我，眼神深邃。
```

**对话融入**
```
"你 calmed（冷静）一点，"他说，"我会 explain（解释）的。"
```

### ❌ 错误示范

**堆砌单词**
```
❌ 我感到 nervous（紧张的）、anxious（焦虑的）、scared（害怕的）...
[太多单词，不自然]
```

**打断阅读**
```
❌ 他走向那个东西（thing，名词，发音/θɪŋ/，意思是"事物"）...
[括号内容太多]
```
