## 🌟 技能介绍

**抖音爆款口播** 是一个专业的短视频脚本生成技能，帮助创作者快速产出**高完播率、高互动率**的口播脚本。

### 核心特点

- ⏱️ **45-60 秒精准时长**：符合抖音黄金时长
- 🎣 **强钩子开头**：3 秒抓住观众注意力
- 💬 **高互动设计**：引导评论、点赞、关注
- 💰 **重利益表达**：直白说省时间/省成本/多赚钱
- 🚫 **去黑话化**：所有术语用大白话解释

## 🎯 脚本结构

### 标准结构（必遵循）

```
0-3 秒：钩子（痛点反问 + 场景冲击）
10 秒：痛点 + 本质（加互动）
20-30 秒：方法 2-3 条（可落地）
5-10 秒：利益点 + 互动
5 秒：总结 + 引导关注
```

### 示例脚本

**主题：门店设备故障快速解决**

```
【0-3 秒 钩子】
门店收银机突然不出小票？员工慌、顾客催，损失全白亏！

【10 秒 痛点 + 本质】
你是不是也遇到过这种情况？收银机不出票、扫码枪没反应，
员工急得团团转，客服还帮不上忙。
本质不是设备笨，是排查方法没找对。

【20-30 秒 核心方法】
记住 3 步，快速解决故障：
1. 先查电源和纸张，排除硬件问题；
2. 重启设备软件，简单又高效；
3. 两种方法找答案，更快更精准。

【5-10 秒 利益点 + 互动】
学会这 3 步，5 分钟解决设备故障，
少亏几百块营业额，还能省人工成本。
评论区告诉我，你门店最常出的故障是什么？

【5 秒 总结 + 引导】
总结起来就是：查硬件、重启、找对方法。
关注我，分享更多门店运维实用技巧！
```

**时长**：约 55 秒  
**字数**：约 220 字

## 🚫 黑话替换清单

本技能严格遵循"去黑话化"原则，所有专业术语都用大白话解释：

| 黑话 | 替换为 |
|------|--------|
| 落地 | 实际用起来 |
| 赋能 | 帮上忙 |
| 闭环 | 完整流程 |
| 链路 | 流程 |
| 垂直场景 | 具体业务场景 |
| 精准检索 | 快速找对应内容 |
| 知识体系搭建 | 整理零散知识 |
| 多模型协作 | 多准备备用 AI 模型 |
| RAG 系统 | 给大模型找资料的图书管理员 |
| 知识卡片拆分 | 把厚手册拆成小知识点卡片 |
| 混合检索 | 两种找卡片的方法一起用 |
| 增量更新 | 定期补充新内容 |

**原则：** 所有专业术语必须用大白话解释，小白能一听就懂。

## 🎭 人设要求

### 语气风格
- ✅ 冷静、清晰、笃定
- ✅ 不夸张、不情绪化
- ✅ 理性派、技术 + 产品双视角

### 常用衔接词
咱们、说直白点、比如、接下来、第一步/第二步、所以、本质、核心、其实、总结起来

### 高频词
本质、核心、方法、步骤、技巧、省钱、省时间、少走弯路、快速解决、实际能用、小白也会

### 禁忌词
❌ 家人们、❌ 宝子们、❌ 低俗梗、❌ 口水化表达

## 🛠️ 使用方法

### 作为使用者

1. **明确主题**：确定要讲什么内容
2. **选择场景**：AI 技术/门店运维/知识变现

4. **调整优化**：根据实际情况微调

4. **替换黑话**：用大白话表达
5. **检查时长**：控制在 45-60 秒
