## 🌟 技能介绍

**Daily Energy** 是一个温暖的每日能量指引技能，结合：

- 📅 **传统干支历法**：GanZhi calendar signals
- 🔮 **参天 AI 八字 MCP**：专业八字分析
- 💪 **积极行动指导**：现代正向 coaching

**核心理念：** 把"运势"当天气参考，而不是宿命！

---

## ✨ 特色功能

### 🌤️ 每日能量卡
- 星级评分（⭐⭐⭐⭐⭐）
- 幸运色推荐
- 宜忌建议（积极正向）
- 今日一句（暖心鼓励）

### 📊 个人档案
- 本地存储（隐私安全）
- 昵称索引
- 出生信息记录
- 五行喜忌标签

### 🕐 定时推送
- 每天早上 8:30 自动推送
- 可配置时间
- 简洁格式（300 字内）

## 🚀 使用方式

### 激活指令

**排盘：**
```
我是 [昵称]，我要排盘
```

**看运势：**
```
我是 [昵称]，我要看运势
```

**示例：**
```
我是嘻嘻公主，我要看运势
```

### 输出示例

```
嘻嘻公主今日能量：⭐⭐⭐⭐✨
幸运色：暖橙色、薄荷绿

今天是乙巳日，木火相生的日子，适合主动出击、链接他人。
宜纳财、交易、会亲友，能量流向"向外拓展"。

卯月春意正浓，你的创造力在升温，但记得先完成再完美。

今日一句：
春风正好，把想做的事轻轻推一把，它会自己往前走。
```
