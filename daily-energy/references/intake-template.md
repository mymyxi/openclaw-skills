# Daily Energy Intake Template

Use this when creating or updating a user profile.

## Short intake version

请把这几项发我，我就能给你建档：
1. 昵称
2. 出生历法：公历 / 农历
3. 出生日期：YYYY-MM-DD
4. 出生时间：尽量精确到小时
5. 性别
6. 出生地
7. 五行喜忌（知道就填，不知道可先空着）
8. 最近最关心的维度：事业 / 健康 / 感情 / 财运 / 家庭

## Gentle clarification rules

- If the time is unknown, ask for an approximate two-hour Chinese hour block if possible.
- If the birth place is vague, ask for city-level detail.
- If the user is unsure whether the birthday is solar or lunar, ask them to confirm before storing.
- Never guess missing fields.

## Tone reminders

- Warm and grounded
- Encouraging, not mystical-pressure
- Practical advice first
