# 本周城市指南 🏙️

通用版城市生活折扣周报生成器，支持任意城市。

## 功能特点

- 🛍️ 购物折扣：商场活动、品牌折扣
- 🍽️ 美食优惠：餐厅活动、周末聚会推荐
- 🌸 游玩娱乐：季节性景点、门票折扣
- 🌍 支持任意城市：北京、上海、深圳...

## 快速开始

```bash
# 指定城市生成周报
node scripts/generate.mjs --city 北京
node scripts/generate.mjs --city 上海
```

## 搜索策略

### 时间筛选规则
- 搜索"本周"活动
- 补充搜索下月初（覆盖跨月）
- 只保留：活动结束时间 ≥ 本周一

### 示例
- 本周：3月16日-3月22日
- 活动A（3.6-4.6）→ 4月6日 ≥ 3月16日 → ✅ 保留
- 活动B（3.1-3.10）→ 3月10日 < 3月16日 → ❌ 删除

## 输出格式

- 购物折扣表格（商场/折扣/时间/来源）
- 美食推荐表格（餐厅/优惠/时间/适合场景）
- 游玩娱乐表格（景点/门票/花期/特色）
- 本周行动建议

## 配置

可选配置文件 `config/cities.json`，自定义城市关键词。
